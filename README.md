# 🛒 E‑Commerce Frontend (Next.js + Redux Toolkit + React Query + Groq AI)

Современный e‑commerce фронтенд с серверным рендерингом, кэшированием данных, AI‑ассистентом и модульной архитектурой.  
Проект разработан с упором на производительность, масштабируемость и чистоту кода.

---

## ✨ Основные возможности

- ⚡ **Next.js App Router** — современная архитектура 
- 🧠 **AI‑поиск, -ассистент на Groq** — быстрые ответы, контекстные подсказки
- 📦 **React Query (TanStack Query)** — кэширование, рефетчинг, оптимистичные обновления  
- 🗂 **Модульная структура** — удобная навигация и масштабирование  
- 🛍 **Категории, товары, подборки, флеш‑распродажи**  
- 🎨 **TailwindCSS** — адаптивный UI  
- 🔐 **Безопасная работа с .env** — приватные ключи только на сервере  
- 🧩 **API‑роуты Next.js** — единая точка входа для данных

## 🔧 Установка и запуск
### 1. Клонировать репозиторий
git clone https://github.com/MrTheSumraak/my-shop/.git
cd my-shop
npm install
Проект использует MongoDB.
Для локального запуска вам необходимо:
Создать собственный кластер MongoDB (например, в MongoDB Atlas).
В файле .env указать строку подключения в переменной: MONGODB_URI
Для корректной работы приложения необходимо создать в кластере Cluster0 базу данных и коллекцию products, куда вы добавите тестовые данные.
Все товары так же лежат в data/allProducts.json
Так же нужен ключ доступа к Groq AI, который должен находиться в .env - GROQ_API_KEY
Запуск проекта: 
npm run dev
